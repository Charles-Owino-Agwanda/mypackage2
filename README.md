# mypackage2

mypackage2 is a Python library for dealing Fibonacci and ordering elements in an array.

## Installation

Use the package manager [pip](https://github.com/Charles-Owino-Agwanda/mypackage2) to install mypackage2.

```bash
pip install git+https://github.com/Charles-Owino-Agwanda/mypackage2
```

## Usage

```python
import mypackage2

# returns '[8,7,5]'
mypackage2.mymodule.top_n([3,7,8,4,5])

```
